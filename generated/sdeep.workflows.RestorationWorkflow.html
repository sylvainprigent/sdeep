sdeep.workflows.RestorationWorkflow#

class sdeep.workflows.RestorationWorkflow(model: SModel, loss_fn: Module, optimizer: Module, train_dataset: SDataset, val_dataset: SDataset, evaluate: SEval, train_batch_size: int, val_batch_size: int, epochs: int = 50, num_workers: int = 0, save_all: bool = False, use_tiling=False)#

Workflow to train and predict a restoration neural network

Parameters:

  • model – Neural network model

  • loss_fn – Training loss function

  • optimizer – Back propagation optimizer

  • train_dataset – Training dataset

  • val_dataset – Validation dataset

  • train_batch_size – Size of a training batch

  • val_batch_size – Size of a validation batch

  • epochs – Number of epoch for training

  • num_workers – Number of workers for data loading

  • use_tiling – use tiling or not for prediction
